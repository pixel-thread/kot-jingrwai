# 🐛 Workflow: Bug Fix

## Trigger

- Bug reported

## Steps

1. Reproduce issue
2. Identify feature domain
3. Locate layer:
   - UI / Hook / API / Service
4. Fix root cause (not symptom)
5. Add test if needed

## Validation

- No regression
- Pass all tests

## Output

- Stable fix
